# 🚀 الميزات المتقدمة لنظام الإعلانات

## ✨ تم إضافة الميزات التالية

### 1. نظام تتبع الإحصائيات 📊

#### **تتبع النقرات والمشاهدات**
- ✅ تتبع تلقائي لكل نقرة على الإعلان
- ✅ تتبع تلقائي لكل مشاهدة (impression)
- ✅ حفظ البيانات في قاعدة البيانات

#### **API Endpoint للتتبع**
```
POST /api/placements/track
```

**Body:**
```json
{
  "adId": "ad-id",
  "action": "click" // or "impression"
}
```

---

### 2. صفحة الإحصائيات الشاملة 📈

#### **الوصول**
```
/admin/promotions/ad-stats
```

#### **المقاييس المعروضة:**
- ✅ **نظرة عامة:**
  - إجمالي المواضع
  - الإعلانات النشطة/الإجمالي
  - إجمالي النقرات
  - إجمالي المشاهدات
  - متوسط معدل النقر (CTR)

- ✅ **أفضل 10 إعلانات:**
  - مرتبة حسب عدد النقرات
  - عرض النوع، المعرّف، الموضع
  - النقرات والمشاهدات
  - معدل النقر لكل إعلان

- ✅ **أداء المواضع:**
  - عدد الإعلانات لكل موضع
  - النقرات والمشاهدات لكل موضع
  - معدل النقر لكل موضع

#### **API للإحصائيات**
```
GET /api/admin/placement-ads/stats
GET /api/admin/placement-ads/stats?placementId=XXX
GET /api/admin/placement-ads/stats?adId=XXX
```

**Response Example:**
```json
{
  "overview": {
    "totalPlacements": 5,
    "totalAds": 15,
    "activeAds": 12,
    "totalClicks": 1250,
    "totalImpressions": 15600,
    "avgCTR": "8.01"
  },
  "topAds": [...],
  "placements": [...]
}
```

---

### 3. دعم الصور والعناوين 🖼️

#### **حقول جديدة في placement_ads:**
- ✅ `title` - عنوان الإعلان (اختياري)
- ✅ `description` - وصف الإعلان (اختياري)
- ✅ `imageUrl` - رابط صورة الإعلان (اختياري)
- ✅ `customData` - بيانات إضافية JSON (اختياري)

#### **واجهة الإضافة المحسّنة:**
- نموذج محسّن لإضافة الإعلانات
- حقول إدخال للعنوان والوصف
- حقل رابط الصورة مع معاينة مباشرة
- التحقق من صحة الصورة

---

### 4. مكون العرض المحسّن 🎨

#### **دعم الصور الكامل:**
```jsx
<AdPlacement location="HOME_TOP" />
```

**يعرض الآن:**
- ✅ صور الإعلانات بجودة عالية
- ✅ عنوان ووصف فوق الصورة (overlay)
- ✅ تصميم responsive
- ✅ تأثيرات hover جذابة
- ✅ fallback للإعلانات بدون صور

#### **أنواع العرض المحسّنة:**

**1. SLIDER / ROTATING:**
- صورة كاملة مع overlay للنص
- دوران تلقائي
- نقاط تحكم يدوي

**2. GRID:**
- صورة بارتفاع ثابت (192px)
- عنوان ووصف أسفل الصورة
- تصميم بطاقة أنيق

**3. CAROUSEL:**
- صور مصغرة (128px)
- عنوان ووصف مختصر
- تمرير أفقي سلس

## 📝 كيفية الاستخدام

### **إضافة إعلان بصورة:**

1. اذهب إلى `/admin/promotions/ad-placements`
2. اختر موضع وافتح "إدارة الإعلانات"
3. انقر "إضافة إعلان جديد"
4. املأ البيانات:
   ```
   نوع المحتوى: AUCTION
   المعرّف: auction-123
   العنوان: مزاد مميز لسيارة BMW 2020
   الوصف: سيارة بحالة ممتازة مع كامل المواصفات
   رابط الصورة: https://example.com/bmw.jpg
   الأولوية: 10
   ```
5. اضغط "إضافة"

### **معاينة الصورة:**
- عند إدخال رابط صورة، ستظهر معاينة مباشرة
- إذا كان الرابط غير صحيح، لن تظهر المعاينة
- يمكنك التأكد من الصورة قبل الحفظ

## 🎯 الإحصائيات المتقدمة

### **معدل النقر (CTR)**
```
CTR = (Clicks / Impressions) × 100
```

### **أفضل الإعلانات**
- مرتبة حسب عدد النقرات
- تساعدك على معرفة أي إعلانات أكثر جذباً
- يمكنك تحسين استراتيجيتك بناءً على البيانات

### **أداء المواضع**
- مقارنة أداء المواضع المختلفة
- معرفة أي مواقع في الموقع أكثر فعالية
- تحسين توزيع الإعلانات
