# قواعد وشروط المشروع - سوق مزاد

## القواعد الصارمة (لا استثناءات)

### 1. التصميم والواجهات
- **ممنوع نهائياً:** استخدام الإيموجي في الكود أو التصميم
- **مطلوب:** استخدام الأيقونات الاحترافية فقط من المكتبات:
  - Heroicons
  - Lucide React
  - React Icons
- **قاعدة:** عدم تغيير التصميم أو الشكل إلا بأمر صريح

### 2. اللغة والتواصل
- **لغة المحادثات:** العربية دائماً
- **لغة المشروع:** العربية
- **المنطقة:** ليبيا
- **العملة:** دينار ليبي (دينار)
- **الأرقام:** الأرقام العادية فقط (123) - لا أرقام هندية (١٢٣)

### 3. التقنيات المستخدمة (لا تغيير)
#### قواعد البيانات:
- **المستخدم:** PostgreSQL + Prisma
- **الممنوع:** SQLite, MySQL, MongoDB

#### التخزين المؤقت:
- **المستخدم:** KeyDB Alternative
- **الممنوع:** Redis

#### التقنيات الأساسية:
- React
- TypeScript
- Next.js
- TailwindCSS
- Monorepo Architecture

### 4. قواعد التطوير
- **لا تشغيل السيرفرات:** المستخدم يقوم بذلك يدوياً
- **الحفاظ على البنية:** عدم تغيير بنية المشروع الأساسية
- **استخدام المكونات الموجودة:** عدم إعادة اختراع العجلة
- **عدم كسر الوظائف:** التأكد من عمل كل شيء قبل التعديل

### 5. معلومات البيئة
- **قاعدة البيانات:** PostgreSQL على localhost:5432
- **KeyDB:** يعمل على localhost:6379
- **التطبيق:** يعمل على localhost:3021
- **Prisma:** متصل ومُعد بالكامل

## الأولويات عند أي عملية

1. **التوافق:** مع PostgreSQL و Prisma
2. **الاستقرار:** عدم كسر الوظائف الحالية
3. **الأداء:** استخدام المكونات المحسنة الموجودة
4. **الأمان:** اتباع معايير الأمان المطبقة
5. **اللغة:** كل النصوص بالعربية

## البنية الأساسية للمشروع

```
c:/sooq-mazad/
├── apps/
│   ├── admin/     # لوحة التحكم الإدارية
│   ├── api/       # API Server
│   └── web/       # الموقع الرئيسي
├── packages/

│   ├── cache/     # نظام KeyDB

├── components/    # مكونات React
├── lib/           # المكتبات والخدمات
├── utils/         # الدوال المساعدة
└── types/         # TypeScript Types

```

## المكونات الرئيسية المستخدمة

- **AdminLayoutWithTransitions:** تخطيط لوحة التحكم
- **SimpleAdminAuth:** نظام المصادقة الإداري
- **CheckboxField:** مكون موحد للاختيار
- **AdminTableAdapter:** جداول إدارية متجاوبة
- **TransportImageUploader:** رفع صور خدمات النقل
- **ComprehensivePhoneSystem:** نظام أرقام الهواتف الليبية

## ملاحظات مهمة

- المشروع يستخدم Turbo للـ build
- النظام يدعم أرقام الهواتف الليبية بالكامل
- جميع APIs محمية بنظام مصادقة
- النظام يدعم Soft Delete للبيانات الحساسة
