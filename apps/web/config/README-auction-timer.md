# تكوين عداد المزايدة الدائري

## نظرة عامة

هذا الملف يحتوي على التكوين المحفوظ والمعتمد لأحجام عداد المزايدة الدائري في
المشروع.

## الملفات

- `auction-timer-sizes.ts` - ملف التكوين الرئيسي
- `SimpleCircularAuctionTimer.tsx` - المكون الذي يستخدم هذا التكوين

## الأحجام المعتمدة

### Small (صغير)

- الحجم: `w-44 h-44` (176px)
- نصف القطر: 75px
- سُمك الخط: 3px

### Medium (متوسط) - **الحجم المحفوظ والمعتمد**

- الحجم: `w-56 h-56` (224px)
- نصف القطر: 98px
- سُمك الخط: 5px
- **هذا هو الحجم الافتراضي للمشروع**

### Large (كبير)

- الحجم: `w-64 h-64` (256px)
- نصف القطر: 110px
- سُمك الخط: 6px

## الاستخدام

```typescript
import {
  getAuctionTimerSizeConfig,
  DEFAULT_AUCTION_TIMER_SIZE,
} from '../config/auction-timer-sizes';

// استخدام الحجم الافتراضي
const sizeConfig = getAuctionTimerSizeConfig();

// استخدام حجم محدد
const mediumConfig = getAuctionTimerSizeConfig('medium');
```

## التحديثات

- **تاريخ الحفظ**: تم حفظ الحجم المتوسط `w-56 h-56` كحجم معتمد
- **السبب**: تحسين التصميم وزيادة وضوح العرض
- **التأثير**: جميع عدادات المزايدة ستستخدم هذا الحجم افتراضياً

## ملاحظات مهمة

1. **لا تقم بتعديل** الأحجام في ملف `auction-timer-sizes.ts` إلا بعد موافقة
   الفريق
2. **الحجم المتوسط** هو الحجم المعتمد والمحفوظ للمشروع
3. **جميع التحديثات** يجب أن تتم عبر ملف التكوين وليس مباشرة في المكونات
4. **التوافق**: هذا التكوين متوافق مع جميع حالات المزاد (قادم/نشط/منتهي)

## الصيانة

عند الحاجة لتحديث الأحجام:

1. قم بتحديث ملف `auction-timer-sizes.ts`
2. تأكد من اختبار جميع الأحجام
3. حدث هذا الملف التوثيقي
4. أعلم الفريق بالتغييرات
